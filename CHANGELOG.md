# Changelog

All notable changes to this project will be documented in this file.

This project follows a simple changelog format inspired by Keep a Changelog.

## [1.0.0] - 2026-06-01

### Added

- Mobile-first ListenWrite English MVP.
- Create practices from YouTube URLs and Arabic subtitle files.
- SRT and VTT subtitle parsing.
- Supabase persistence for practices, answers, replay counts, progress, and difficult markers.
- YouTube segment playback, replay, slow playback, pause, caption toggle, and custom timeline.
- Segment list with timestamps, active segment auto-scroll, completion color, and auto-play on selection.
- Sticky mobile controls for Previous, Next, and Mark as Difficult.
- Vercel deployment support.

### Security

- Public environment example uses placeholders only.
- Local environment files, Vercel metadata, build output, and dependencies are ignored by git.
